Training and deploying a DL model

Date: 01/03/2024

This is my first attempt of creating and training a model on my own machine by using a linux environment on windows which was a task in itself. After managing to get all that up and running with the necessary dependencies, I made use of Jupyter and the notebook feature to test and train my model.

The model uses a great dataset I found that represents categories of music into spectrograms. I fine tuned my model doing 80 epochs because I began to find diminishing returns and wanted to get my model on and running.

The model has 80.5% accuracy rate and tends to get confused with hip-hip and rock based on the confusion matrix. I would have liked this to be better in terms of accuracy but decided that I don't wanna waste my time anymore on training.

Here is an example prediction of a country song, the model being quite sure its a country song. At least deep learning models like country music, because I don't think anyone else does.

The next step was to implement the model onto Hugging Face to host. After some (many) google searches later, I finally got a working python code to deploy the model on Hugging Face using gradio.

Hugging Face Space

I tested the hugging space with a 30sec extract from Jay-Z's 'The Story of OJ' which is very much a hip-hop song.

Final Thoughts:

I hope to implement this into this website by using the Hugging Face API, and hopefully allowing input to be a spectrogram image, a 30 second audio file, or to be able to use a microphone to hear music being played.

Diving into AI and Deep Learning

Date: 28/02/2024

I have always wanted to delve into AI and machine learning but was always daunted by the task of it but luckily I came across a great learning resource by fast.ai.

Practical Deep learning course by fast.ai

So far I am up to lesson 2 of the course, and hope to deploy a my own macine learning project. Although quite simple, this is my first dive into deep learning.
